# 📊 Sales Performance Analytics Dashboard

## Project Overview

This project presents an interactive Sales Performance Analytics Dashboard designed to analyze business performance through key sales metrics, trend analysis, and data visualization techniques. The dashboard helps identify sales patterns, customer behavior, and category performance to support data-driven decision making.

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Dashboard Design
- Data Visualization
- Business Analytics



## Key Performance Indicators (KPIs)

- Average Order Value
- Weekly Average Number of Orders
- Number of Orders by Country
- Number of Orders by Category
- Moving Average Trends
- Time-Based Sales Analysis


## Dashboard Features

- Interactive filtering options
- Moving average trend analysis
- Country-wise order distribution
- Category-wise performance analysis
- KPI visualization
- Time-period comparison



## Business Insights

- Identified fluctuations in average order value over time.
- Analyzed weekly order volume trends.
- Compared sales performance across multiple countries.
- Evaluated category contribution to total orders.
- Generated insights for improving business performance and decision-making.
